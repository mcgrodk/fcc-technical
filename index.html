Preface

This lesson will introduce some foundational concepts of Japanese grammar. First we will learn about the building blocks of Japanese grammar that form sentences, and then we will learn how to form some basic sentences.

Prerequisites: The reader should be able to read Hiragana and Katakana to follow this lesson, as Romaji will not be used with Japanese sentences. Kanji characters will not be used in this lesson.

See Tae Kim's Guide to Japanese for a lesson on the Hiragana and Katakana writing systems.

Japanese Grammar Particles

The Japanese language uses a system of grammar "particles" that indicate the role of different items in a sentence, such as subject, object, location, and date/time. These particles will be explained with further context in later sections of this lesson, but you can refer to the paragraphs below for a simple explanation of each particle and the role it fills in a sentence.

Subject Marker 「は」

The particle 「は」marks the subject of a sentence, so it is usually the first particle to be used in a complete Japanese sentence. Despite being written as 「は」, the character is pronounced like 「わ」whenever it is used as this subject marker particle.

わたしはがくせいです。
I am a student.
かのじょはにほんじんです。
She is Japanese.

A note on "null-subject" sentences

Japanese is a "null-subject language," which means there are cases when the subject of a sentence can be omitted while still being gramatically correct.
